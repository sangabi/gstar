## 📝 똑똑노트 Pro 사용 설명서

**똑똑노트 Pro**는 개인 보안과 편리한 메모 관리에 최적화된 나만의 프라이빗 메모장 앱입니다.

### 1. 시작하기 및 보안 설정

* **첫 실행 및 비밀번호**: 앱을 처음 실행하면 비밀번호 설정 화면이 나타납니다.
* **보안 유지**: 4자리 이상의 숫자 비밀번호를 입력하여 나만의 암호를 설정하세요.
* **잠금 해제**: 이후 앱을 열 때마다 설정한 비밀번호를 입력해야만 메모 내용이 나타납니다.

### 2. 메모 작성 및 이미지 첨부

* **내용 입력**: 상단 입력창에 자유롭게 메모를 작성하세요.
* **카테고리 분류**: '일반', '업무', '아이디어', '중요' 중 하나를 선택하여 메모의 성격을 지정할 수 있습니다.
* **이미지 추가**: [파일 선택] 버튼을 눌러 스마트폰에 있는 사진을 메모와 함께 저장할 수 있습니다.
* **저장**: 모든 입력을 마친 후 [저장] 버튼을 누르면 목록 맨 위에 메모가 추가됩니다.

### 3. 메모 관리 및 검색

* **실시간 검색**: 중간의 돋보기 모양 검색창에 단어를 입력하면 메모 내용이나 카테고리 이름으로 메모를 즉시 찾아줍니다.
* **메모 삭제**: 더 이상 필요 없는 메모는 우측 상단의 빨간색 [삭제] 버튼을 눌러 지울 수 있습니다.

### 4. 부가 기능 이용하기

* **다크 모드 (🌓)**: 상단 우측의 반달 아이콘을 누르면 눈이 편안한 다크 모드로 화면 테마를 바꿀 수 있습니다.
* **백업 (내보내기)**: [백업] 버튼을 누르면 지금까지 작성한 모든 메모가 텍스트 파일(`.txt`)로 변환되어 휴대폰에 저장됩니다. 중요한 내용은 주기적으로 백업하세요.

### 💡 알아두면 좋은 팁

* **데이터 저장**: 작성하신 메모는 스마트폰 내부 저장소(`LocalStorage`)에 안전하게 저장됩니다.
* **주의사항**: 앱의 캐시를 삭제하거나 데이터를 모두 지우면 저장된 메모가 삭제될 수 있으니, 중요한 내용은 반드시 **백업 버튼**을 활용해 따로 보관해 주세요.
